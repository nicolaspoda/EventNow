# EventNow — Frontend

Application React + TypeScript + Vite du projet EventNow. Voir le [README racine](../README.md) pour la présentation du projet, la stack complète et les instructions de lancement (Docker).

## Commandes utiles (hors Docker)

```bash
npm install
npm run dev            # serveur de dev
npm run build           # build de production
npm run test            # tests unitaires (Vitest)
npm run lint
```
