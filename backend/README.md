# EventNow — Backend

API NestJS du projet EventNow. Voir le [README racine](../README.md) pour la présentation du projet, la stack complète et les instructions de lancement (Docker).

## Commandes utiles (hors Docker)

```bash
npm install
npm run start:dev   # mode watch
npm run test        # tests unitaires
npm run test:e2e    # tests e2e
npm run lint
```
